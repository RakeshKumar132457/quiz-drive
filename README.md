# Car Game and Custom Reports Application

A TypeScript application featuring a car racing game with quiz integration and a custom reports generator.

## Features

### Car Racing Game
- Interactive car movement with keyboard and button controls
- Dynamic road rendering
- Quiz integration with true/false questions
- Collision detection system

### Custom Reports
- Dynamic metric selection
- Interactive data visualization using Chart.js
- Multiple chart types (Bar and Line)
- CSV export functionality
- Email report capability
- Customizable filters for each metric

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

### Car Game
- Use arrow keys or on-screen buttons to move the car left and right
- Avoid obstacles and answer quiz questions
- Score points by providing correct answers

### Reports Module
1. Navigate to the Reports section
2. Select desired metrics from the available options
3. Choose appropriate filters for each metric
4. Select chart type (Bar or Line)
5. Click "Generate Report" to visualize data
6. Use "Export CSV" to download data
7. Use "Email Report" to share reports
